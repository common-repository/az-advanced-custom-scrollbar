== Description ==
"AZ Advanced Custom Scrollbar" - is a modern custom scrollbar, which allows you to replace the existing scrollbar of your website by an awesome custom scrollbar. The scrollbar is extremely smooth and consistent in modern browsers and mobile devices, with low resource usage.

== Installation ==
This section describes how to install the "AZ Video and Audio Player Addon for Elementor" plugin and get it working.

= 1) Install =

i. Go to the WordPress Dashboard "Add New Plugin" section.

ii. Search For "AZ Advanced Custom Scrollbar".

iii. Install, then Activate it.

= OR: =

i. Unzip (if it is zipped) and Upload `az-advanced-custom-scrollbar` folder to the `/wp-content/plugins/` directory

ii. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
i. After activate the plugin. It will replaced the default scrollbar instantly

ii. For customization navigte to "Settings" > "AZ Scrollbar Settings"

iii. Play with the options, Save and relax!
